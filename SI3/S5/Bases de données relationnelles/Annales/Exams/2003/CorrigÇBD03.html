PC(model, speed, ram, hd, cd, price)


Écrire la requête SQL qui permet d'obtenir tous les fabricants (i.e., maker) de PC dont la vitesse est au moins supérieure ou égale à 1600 (donner deux versions: l'une avec WHERE â€¦ IN et l'autre avec WHERE EXISTS)


Réponse:

SELECT maker

FROM Product

WHERE model IN

(SELECT model

FROM PC

WHERE speed >= 1600);


SELECT maker

FROM Product

WHERE EXISTS

(SELECT *

FROM PC

WHERE speed >= 1600 AND

Product.model = model);

  1. Soit le schéma suivant :


Product(maker, model, type)

PC(model, speed, ram, hd, cd, price)

Laptop(model, speed, ram, hd, screen, price)

Printer(model, color, type, price)


      1. Écrire une requête qui affiche pour chaque fabricant (i.e., maker) la taille moyenne des écrans de ses ordinateurs portables (i.e., Laptop)


      1. Écrire une requête qui affiche le prix moyen pour chaque vitesse de PC supérieure à 150.

Réponse:

SELECT maker, AVG(screen)

FROM Product, Laptop

WHERE Product.model = Laptop.model

GROUP BY maker;

SELECT speed, AVG(price)

FROM PC

WHERE speed > 150

GROUP BY speed;

On remarquera que la propriété au sujet de "speed " n'est pas une propriété d'un groupe et il n'y a donc pas besoin de clause HAVING.

  1. A partir des tables suivantes :


MovieStar(name, address, gender, birthdate)

MovieExec(name, address, cert#, networth)

StudioPres(name, address) % Coordonnées des présidents de studio


      1. Construire une vue RichExec donnant le nom, l'adresse, le numéro de certificat et la valeur networth de tous les MovieExec dont ladite valeur est supérieure à $100000

      2. Utiliser cette vue pour écrire une requête qui affiche les noms des MovieExec qui sont aussi présidents de studio.



Réponse:

CREATE VIEW RichExec AS

SELECT *

FROM MovieExec

WHERE netWorth >= 100000;


SELECT RichExec.name

FROM RichExec, StudioPres

WHERE RichExec.name = StudioPres.name;



  1. Expliquer le rôle exact du serveur RESIN lors du chargement d'un page jsp (e.g. la page marques.jsp dans le TP5) suite à une requête d'un serveur Web


Réponse:

Le serveur compile et exécute le code JAVA, envoie le cas échéant au serveur de base de données les requêtes générées à partir du code JDBC, génère le code HTML correspondant aux affichages produits et retourne ce code au serveur Web.
